# ICA — Desarrollo Web & Soluciones Digitales

Sitio web de [icaweb.dev](https://icaweb.dev), estudio digital de Imanol Castro.

## Stack

- [Astro](https://astro.build/) — Framework estático con islands architecture
- [React](https://react.dev/) — Componentes interactivos (Navbar, Servicios, Contacto)
- [Tailwind CSS v4](https://tailwindcss.com/) — Estilos utilitarios
- [Lucide React](https://lucide.dev/) — Iconografía

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
